# hear-cut

나주혁신도시(빛가람동) 미용실을 모바일에서 빠르게 비교하고, 최종 예약은 기존 네이버 예약/플레이스 링크로 넘기는 비교 가이드 웹앱 초안이다.

## 목표

- 나주혁신도시 미용실을 거의 전체 수록
- 가격대, 주차, 1인샵, 남성커트, 펌, 염색, 네이버예약 가능 여부를 한눈에 비교
- 모바일 퍼스트 UI/UX
- 자체 예약 시스템 없이 외부 예약 링크 연결

## 기술 방향

- Next.js App Router
- 정적 페이지 기반
- 이후 Supabase를 데이터 원본으로 연결 가능
- GitHub 연동 후 Vercel 배포 예정

## 로컬 실행

의존성은 아직 설치하지 않았다. 배포 또는 실행 시 아래 명령으로 설치 후 실행한다.

```bash
npm install
npm run dev
```

## 현재 포함된 것

- 홈 화면
- 미용실 카드 목록
- 상세 페이지 동적 라우트
- 초기 데이터 샘플
